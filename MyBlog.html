UNIX COMMANDS

* I was practicing the Unix commands and i have found an interesting command.

Hiding :

To create a file/directory as a hidden file/directory , just add dot (.) operator
in front of the file name. Thats gives you the hidden file.


Example:

$touch .filename.txt # This is example to create the hidden file $mkdir .directory # This is example to create the hidden directory

To view the hidden files in unix, $ls -al #This displays all the hidden files/directories
